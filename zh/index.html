    vuepress-plugin-md-enhance

    为 VuePress2 提供更多 Markdown 增强功能

    快速上手 💡配置 🛠

    开箱即用

    • 链接检查

      检查 Markdown 链接

    • GFM 支持

      支持 GitHub 风格 Markdown

    标记内容

    • 自定义容器支持

      用样式装饰 Markdown 内容

    • 自定义属性支持

      让你为 Markdown 元素添加属性

    • 标记支持

      让你在 Markdown 中对词句进行标记

    • 样式化字符

      以你想要的方式样式化字符

    创作内容

    • 自定义对齐支持

      让你在 Markdown 中对段落进行对齐

    • 脚注

      你的 Markdown 现在将支持脚注

    • 导入文件支持

      将你的文档分段，并在 Markdown 中导入

    • 扩展图片语法

      指定图片大小与颜色模式，并将它们转化为插图

    丰富你的内容

    • Tex 支持

      Markdown 现在也可以支持 Tex 语法以显示公式

    • 幻灯片支持

      通过 Reveal.js 在 Markdown 中插入幻灯片

    • 选项卡

      使用选项卡对相似内容进行分组

    • 上下角标支持

      你的 Markdown 现在将支持上下角标

    • 任务列表

      在 Markdown 中使用任务列表

    • 卡片支持

      在 Markdown 中展示卡片

    在 Markdown 中嵌入图表

    • 图表支持

      在 Markdown 中展示图表

    • ECharts 支持

      在 Markdown 中展示 ECharts 图表

    • 流程图支持

      可以在 Markdown 中直接写出流程图

    • Mermaid 支持

      可以在 Markdown 中添加 Mermaid 图例

    展示你的代码和工作

    • 代码选项卡

      使用选项卡对相似代码进行分组

    • 代码案例支持

      你可以很方便的插入代码案例

    • 交互演示支持

      你可以在 Markdown 中添加交互演示

    • Vue 交互演示支持

      在交互演示中展示 Vue 组件

    # 安装

    pnpm
    pnpm add -D vuepress-plugin-md-enhance
    yarn
    yarn add -D vuepress-plugin-md-enhance
    npm
    npm i -D vuepress-plugin-md-enhance

    # 使用

    TS
    // .vuepress/config.ts
import { mdEnhancePlugin } from "vuepress-plugin-md-enhance";

export default {
  plugins: [
    mdEnhancePlugin({
      // 你的选项
    }),
  ],
};
    JS
    // .vuepress/config.js
import { mdEnhancePlugin } from "vuepress-plugin-md-enhance";

export default {
  plugins: [
    mdEnhancePlugin({
      // 你的选项
    }),
  ],
};
